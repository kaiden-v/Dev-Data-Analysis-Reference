# Statsmodels Reference

## Introduction

Statsmodels is a Python package for **statistical modeling and hypothesis testing**. It provides classes and functions for estimating **statistical models**, performing **statistical tests**, and conducting **data exploration**. Statsmodels is commonly used for **regression analysis, time series analysis, categorical data analysis, and statistical inference**. This guide serves as a quick reference to the main features, processes, and functions of the Statsmodels package. As this is a quick reference guide, more in-depth information can be found on the official [Statsmodels Documentation](https://www.statsmodels.org/stable/index.html) page.

## Installing Statsmodels

You can install Statsmodels in your Python environment using the following command in a command line:

```bash
pip install statsmodels <optional_version>
```
To check the current version use

```bash
pip show statsmodels
```

---

## Importing Statsmodel

The standard convention to import Statsmodel into a python file is:

```python
import statsmodels.formula.api as smf
```
