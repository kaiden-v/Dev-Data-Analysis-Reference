# Matplotlib Reference

## Introduction

Matplotlib is a Python library for **creating static, animated, and interactive visualizations**. It provides a flexible framework for plotting **line plots, scatter plots, bar charts, histograms, and more**. Matplotlib is designed to work seamlessly with **NumPy arrays and Pandas DataFrames**, making it a core tool for data analysis and visualization. This guide serves as a quick reference to the main features, processes, and functions of the Matplotlib package. For more in-depth information, see the official [Matplotlib Documentation](https://matplotlib.org/stable/contents.html) page.

## Installing Matplotlib

You can install Matplotlib in your Python environment using the following command:

```bash
pip install matplotlib <optional_version>
```
To check the current version use

```bash
pip show matplotlib
```

---

## Importing Matplotlib

The standard convention to import matplotlib plotting class into a python file is:

```python
import matplotlib.pyplot as plt
```
